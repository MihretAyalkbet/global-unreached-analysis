#  Unreached & Frontier People Groups Dashboards

This project contains two Tableau dashboards visualizing the global distribution of Unreached and Frontier People Groups using data from the Joshua Project.

---

##  Dashboard 1: Top 10 Unreached Countries & Mission Focus

**What it shows:**
- Bar chart of the **Top 10 Most Unreached Countries** by total population.
- Bubble chart highlighting nations with the **greatest need for Gospel workers**.
- A region filter (Africa, Asia, etc.) to focus mission attention.
- Interactive visuals that allow users to compare global unreached population patterns.

**Purpose:**
To identify the countries with the largest unreached populations and highlight strategic mission priorities.

---

##  Dashboard 2: Global Frontier & Unreached Map

**What it shows:**
- A world map plotted using **Latitude & Longitude** for each people group.
- Circle size = **population size**.
- Color = **Frontier** vs **Unreached** groups.
- Visual context and a short reflection on Luke 10:2.
- Highlights major regions with concentrated mission needs (South Asia, Middle East, Africa).

**Purpose:**
To provide a visual global overview of unreached people groups and highlight where missionary presence is most needed.
